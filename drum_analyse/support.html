常见问题

FAQ

01 · 如何订阅 / 取消 鼓分析 Pro？

在 App Store 账户的「订阅」中管理订阅，或在应用内前往 Pro 升级页查看当前订阅状态。取消后，你仍可在当期订阅期间继续使用 Pro 功能，到期后将自动停止续费。

02 · 如何申请退款？

退款由 Apple 单独决定，请通过 App Store · 账户 · 购买记录 · 报告问题 提交申请。本应用开发者无法直接处理退款。

03 · 识别结果不准确怎么办？

请确保原音频清晰、节拍可辨，避开 BGM 复杂或失真较大的片段；若仍不理想，可在识别结果上手动微调节拍位置，再保存为 PRESET 重复使用。

04 · PRESET 如何同步到其他设备？

PRESET 默认保存在本机。在 iOS 16 / iPadOS 16 及以上且开启 iCloud Drive 的设备上，可通过 设置 · Apple ID · iCloud · iCloud Drive 开启 鼓分析 的同步权限。

05 · 变速练习如何使用？

在播放器中拖动速度滑块，或点击预设档位（0.5x / 0.75x / 1.0x / 1.25x），即可实时改变播放速度，节拍网格与节拍提示音会同步调整。

06 · Humanize（人性化）是什么？

同时对鼓点的时机与力度施加随机偏移，让量化过度的 Pattern 更接近真人演奏，参考 Logic Pro MIDI Transform Humanize 设计。在 传输栏 · HUM 中调节强度。

03

故障排除

Troubleshooting

没有声音 / 声音很小

  1. 检查设备静音开关和系统音量；
  2. 确认未连接蓝牙耳机或外放未切换到其他输出设备；
  3. 关闭可能占用音频路由的应用（通话、录屏、其他播放器）；
  4. 重启 鼓分析，必要时重启设备。

应用卡顿 / 性能问题

  1. 关闭后台不用的应用，释放内存；
  2. 检查设备剩余存储，建议保留至少 500 MB 可用空间；
  3. 在 设置 · FX 中关闭不必要的实时效果；
  4. 升级到最新版本的 鼓分析 与系统。

识别 / 生成失败

  1. 确认设备已联网（识别与生成需要在线服务）；
  2. 检查文件格式（建议 WAV / AAC / MP3，采样率 44.1 kHz 及以上）；
  3. 避开受数字版权保护 (DRM) 的音频；
  4. 若长时间无响应，请重启应用后再试。

订阅状态不更新 / Pro 功能未解锁

  1. 打开 App Store · 账户 · 购买记录，确认订阅已生效；
  2. 在 鼓分析 内前往 设置 · 恢复购买 刷新订阅状态；
  3. 退出并重新登录 Apple ID；
  4. 若仍异常，请通过上方邮箱联系我们，并附上订单号。
