二、常见问题 (FAQ)

1. 如何订阅 / 取消 Drummy Pro？

您可以在 App Store 账户的「订阅」中管理订阅，或在应用内前往 Pro 升级页查看当前订阅状态。取消后，您仍可在当期订阅期间内继续使用 Pro 功能，到期后将自动停止续费。

2. 如何申请退款？

退款由 Apple 单独决定，请通过 App Store > 账户 > 购买记录 >「报告问题」提交申请。本应用开发者无法直接处理退款。

3. 识别结果不准确怎么办？

请确保原音频清晰、节拍可辨，避开 BGM 复杂或失真较大的片段；若仍不理想，可在识别结果上手动微调节拍位置，再保存为 PRESET 重复使用。

4. PRESET 如何同步到其他设备？

PRESET 默认保存在本机。在 iOS 16 / iPadOS 16 及以上且开启 iCloud Drive 的设备上，可通过「设置 > Apple ID > iCloud > iCloud Drive」开启 Drummy 的同步权限，以在多台设备间同步。

5. 变速练习如何使用？

在播放器中拖动速度滑块，或点击预设档位（0.5x / 0.75x / 1.0x / 1.25x），即可实时改变播放速度，节拍网格与节拍提示音会同步调整。

6. 误触购买后能取消吗？

可以。请尽快在 iOS 设备的「设置」>「[您的姓名]」>「订阅」中找到对应订阅项并取消；如需退款，请通过 App Store 官方渠道提交申请。

三、故障排除

没有声音 / 声音很小

  1. 检查设备静音开关和系统音量；
  2. 确认未连接蓝牙耳机或外放未切换到其他输出设备；
  3. 关闭可能占用音频路由的应用（通话、录屏、其他播放器）；
  4. 重启 Drummy，必要时重启设备。

应用卡顿 / 性能问题

  1. 关闭后台不用的应用，释放内存；
  2. 检查设备剩余存储，建议保留至少 500 MB 可用空间；
  3. 在应用内关闭不必要的实时效果；
  4. 升级到最新版本的 Drummy 与系统。

识别 / 生成失败

  1. 确认设备已联网（识别与生成需要在线服务）；
  2. 检查文件格式（建议 WAV / AAC / MP3，采样率 44.1 kHz 及以上）；
  3. 避开受数字版权保护 (DRM) 的音频；
  4. 若长时间无响应，请重启应用后再试。

订阅状态不更新 / Pro 功能未解锁

  1. 打开 App Store > 账户 > 购买记录，确认订阅已生效；
  2. 在 Drummy 内前往「设置 > 恢复购买」刷新订阅状态；
  3. 退出并重新登录 Apple ID；
  4. 若仍异常，请通过上方邮箱联系我们，并附上订单号。

四、系统与设备要求

  • 系统版本：iOS 16.0 / iPadOS 16.0 / macOS 13.0 及以上（Mac Catalyst）。
  • 设备：iPhone / iPad / Mac（Apple Silicon 或 Intel）。老旧机型可能存在功能限制。
  • 存储空间：建议保留至少 500 MB 可用空间，用于本地缓存与 PRESET。
  • 网络：识别、生成等在线功能需要稳定的网络连接；本地播放与编辑离线可用。
  • 权限：麦克风（用于录制与识别，仅在您主动启用时申请）；媒体库 / 文件（用于导入您设备中的音频文件）。
